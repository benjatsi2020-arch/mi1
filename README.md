# Suite de Herramientas de Ciberseguridad e Investigación

Este repositorio contiene un conjunto de scripts de automatización, reportes de auditoría de red y diccionarios de claves desarrollados de forma local en un entorno de Kali Linux (WSL).

## 📁 Contenido del Laboratorio

* **`auditoria_scanme.txt`**: Reporte técnico completo de reconocimiento de red y enumeración de servicios web y puertos realizado con Nmap.
* **`creador.py`**: Script automatizado en Python diseñado para generar reportes locales de forma secuencial.
* **`lector.py`**: Programa en Python estructurado para abrir, limpiar y leer bases de datos de texto línea por línea.
* **`diccionario_prueba.txt`**: Diccionario matemático de combinaciones de claves de 4 caracteres generado mediante Crunch.
* **`40142.php` y `45233.py`**: Copias de investigación teórica de pruebas de concepto (PoC) analizadas con Searchsploit.
